- EDA
- Feature engineering
- ARIMA / SARIMA / SARIMAX baselines
- Ridge / Random Forest / XGBoost
- Optuna tuning
- TimeSeriesSplit validation

- FastAPI real-time forecasting
- Streamlit dashboard
- Docker + Render deployment
- CI/CD
